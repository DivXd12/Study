# Diagnostic Medical Simplificat

# Descriere

Aplicație desktop realizată în Python care simulează un sistem simplificat de diagnostic medical pe baza simptomelor introduse de utilizator. Interfața este realizată cu Tkinter.

# Funcționalitate

Utilizatorul răspunde la o serie de întrebări (da/nu). Pe baza răspunsurilor, aplicația calculează probabilități pentru mai multe boli și afișează rezultatul final.

# Boli analizate

* Raceala
* Gripa
* COVID-19
* Alergie
* Bronsita
* Pneumonie

# Metodă

Fiecare simptom modifică o valoare asociată fiecărei boli. La final, valorile sunt normalizate și transformate în procente.

# Tehnologii

* Python 3
* Tkinter

# Limitări

Aplicația este doar un model educațional. Nu oferă diagnostic real și nu folosește date medicale.
